# IMPLEMENTATION CONTRACT

Aquest contracte defineix el format d’output obligatori de l’agent Implementer.

L’agent ha de retornar sempre aquest format:

## TASK
Identificador i nom de la task executada.

## FILES_CHANGED
Llista de fitxers creats o modificats.

## CHANGES_MADE
Resum concret del que s’ha implementat.

## VALIDATION
Com s’ha validat que la task compleix l’objectiu.

## NOTES
Bloquejos, dubtes, limitacions o decisions menors detectades.
